Installation
================

Prerequisites

	# node.js npm
	# brunch.io

Compile scss
	
	./compile.sh


sass and compass
----------
	
для работы нужен sass и compass
	
	gem install sass 
	gem install compass

файл с настройками compass 
	
	config.rb
